## Live Preview

You can double click the block to edit its contents.

<InteractiveBlockPreview symbolId={textSymbolId} />

## Configuration

The text block accepts many `#` commands for styling text, such as
`#text-red-600 #font-bold`:

<InteractiveBlockPreview
  symbolId={textSymbolId}
  blockText="Cats are great pets. They make loyal and loving companions who can provide years of fun and joy. #text-red-600 #font-bold"
/>

## Limitations

There is currently no way to change the font size or font family.
