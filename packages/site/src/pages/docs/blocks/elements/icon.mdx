## Configuration

Type within the icon block to search
[iconify](https://icon-sets.iconify.design/?query=twitter) for an icon.

The icon block will use the top recommended search result automatically, e.g.
for `twitter`:

<InteractiveBlockPreview symbolId={iconSymbolId} blockText="twitter" />

You can also search [iconify](https://icon-sets.iconify.design/?query=twitter)
yourself in another browser tab and use a specific icon name directly, e.g.
`iconoir:twitter-verified-badge`:

<InteractiveBlockPreview
  symbolId={iconSymbolId}
  blockText="iconoir:twitter-verified-badge"
/>

You can configure the icon color using a `#fill-` command, such as
`#fill-blue-400`:

<InteractiveBlockPreview
  symbolId={iconSymbolId}
  blockText="twitter #fill-blue-400"
/>
