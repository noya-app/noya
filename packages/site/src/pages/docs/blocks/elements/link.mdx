## Configuration

The text block accepts many `#` commands for styling text, such as
`#text-red-600 #font-bold`. You can also use the `#icon-arrow-forward` command
to add an arrow icon:

<InteractiveBlockPreview
  symbolId={linkSymbolId}
  blockText="View all cats #text-red-600 #font-bold #icon-arrow-forward"
/>
